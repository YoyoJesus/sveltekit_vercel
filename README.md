# KHE 2026 Early Sign-Up

## Technologies
- Svelte
- Vite
- Bootstrap

## About
Basic early sign-up page for KHE.io for KHE 2026. Project base was generated via:
```bash
npm create vite@latest project_name -- --template svelte
```

## Build :wrench:
```bash
cd khe2026_early_signup
npm install
npm run dev
```
